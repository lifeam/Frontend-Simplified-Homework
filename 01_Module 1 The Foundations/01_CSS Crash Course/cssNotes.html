Methods for Adding CSS

Methods for adding CSS

  • Inline CSS

    • Style directly in the HTML element

    • <h1 style="color: red">

  • Internal CSS

    • Using <style> tag within an HTML file

    • Goes in the <head> tag within an HTML file

      • <style> h2 { color: purple; } /* target the <h2> element */ </style>

  • External CSS (recommended)

    • Linking an external CSS file

    • Must create separate file and post the link in the <head> tag

      • <link rel="stylesheet" href="./style.css">
